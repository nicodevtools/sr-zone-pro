## What It Does

Auto-detects and draws key support & resistance zones based on real price action — not just random lines:

| Feature | Detail |
|---------|--------|
| 📊 **Auto S/R Detection** | Pivot-based with configurable period |
| 🎚️ **Major/Minor Zones** | Colored by zone strength (test touches) |
| 📦 **Zone Boxes** | Visual boxes + horizontal lines |
| 🔔 **Price Touch Alerts** | Get notified when price hits a zone |
| 🎨 **Heikin Ashi Option** | Smoother zones for ranging markets |

## Installation

1. TradingView → Pine Editor
2. Paste `sr-zone-pro.pine`
3. Add to chart
4. Adjust `Pivot Period` to match your style (20=swing, 10=intraday, 5=scalp)

---

## Settings Guide

| Setting | Recommended |
|---------|-------------|
| Pivot Period | 10-20 (higher = stronger zones) |
| Zone Strength | 3 (shows zones tested 3+ times) |
| Minor Zones | ON for day trading |
| Heikin Ashi | ON for crypto, OFF for forex |
